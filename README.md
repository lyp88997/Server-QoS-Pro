# Server-QoS-Pro
一款基于原生 Linux TC &amp; iptables 开发的智能端口流量监控与动态限速工具。它能自动识别异常流量高压，并在设定的时间后自动解除限制，实现全自动的服务器带宽优化管理。
• ⚡ 智能触发限速：支持自定义“带宽阈值 × 持续时间”触发规则。例如：当 443 端口流量超过 80Mbps 持续 5 分钟时，自动下放限速指令。

• 🔄 循环自动恢复：限速时长到期后，系统会自动撤销限制并重新进入监控状态，无需人工干预。

• 🛠️ 中文交互式菜单，支持规则的增删改查、网卡切换及采样频率自定义。
• 📦 生产级稳定性：

• 环境自适配：首次运行自动检测并补齐 tc, iptables, bc, ifstat 等核心依赖。

• 持久化运行：支持一键安装为 systemd 系统服务，实现开机自启。

• 全栈兼容：采用 iptables mangle 标记技术，完美兼容 IPv4/IPv6 以及 NAT 复杂的网络环境。

• 📊 透明化监控：实时显示每个端口的流量状态、限速倒计时以及底层 TC 队列运行情况。
